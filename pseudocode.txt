To plot subset accuracy of data
	- Get true variables (1, 7, 12)
	- For each number of samples n (x-axis)
		- Generate n samples based on data distribution
		- Find optimal lasso parameter
		- Create lasso model and compare with true variables
		
Ways to find optimal lasso parameter:
	- Min MSE CV
		- Almost always overfits
	- Min prediction MSE
		- Almost always overfits
	- Min Bayseian Information Criteria (BIC)
		- Almost always overfits
	- Min Prediction CV MSE + one standard dev (tradeoff variance for bias)
		- Is ok? not great
	- Min MSE CV + One standard dev
		- It's ok too
	- Non-negative Garotte
	- Adaptive Lasso/LARS?
	- SCAD
	- Dantzig
	- VISA
